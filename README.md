## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- PHP 7.0+ (for experiments 8 & 9)
- Web server (Apache, Nginx, or PHP built-in server)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/web-technologies.git
   cd web-technologies
   ```

2. **Static Experiments (1-7)**
   Simply open `index.html` in your browser or serve via any HTTP server:
   ```bash
   # Using Python
   python3 -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```

3. **PHP Experiments (8-9)**
   ```bash
   # Using PHP built-in server
   cd experiment8
   php -S localhost:8000
   ```

## 🩸 RaktSetu: Full-Stack Project

A complete full-stack emergency blood donor network platform built with HTML5, CSS3, Vanilla JavaScript, PHP 8.0+, and MySQL.

### Project Overview
RaktSetu is an automated SOS platform that connects blood donors with patients in emergency situations. When an urgent blood request is raised, the system instantly filters eligible donors by blood group and location, with built-in safety protocols.

### Technologies Used
- **Frontend:** HTML5, CSS3, Responsive JavaScript
- **Backend:** PHP 8.0+ with PDO
- **Database:** MySQL with normalized schema
- **Real-Time Updates:** AJAX polling
- **Security:** Password hashing, prepared statements, role-based access control

### Key Features
- Smart donor-patient matching by blood group and location
- Real-time emergency dashboard with live updates
- Multi-user portal system (Donors, Hospitals, Admins)
- Donor eligibility tracking system
- Secure authentication and data handling

**Location**: `/project/` folder

**Files**: Complete full-stack application with dashboard, authentication, and database integration
